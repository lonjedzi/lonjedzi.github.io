## Active Directory Lab

Active Directory (AD) is a critical tool for managing networks, users, and devices, and as a Cyber Security Analyst, deeply understanding this technology is essential for identifying misconfigurations, managing access controls, and enhancing network security.

In this series, I am demonstrating how I set up and manage Active Directory on Windows Server 2022. This project showcases my ability to configure and optimize AD while highlighting its importance in secure network management.

### What Will Be Covered
Part 1: Install Active Directory on Windows Server 2022.
Part 2: Setting up a NAT network, update the server, configure a static IP, enable Remote Desktop, and rename the server.
Part 3: Transforming the server into a domain controller and joining it to a domain.
Part 4: Configuring DNS forward lookup zones.
Part 5: Joining a Windows 11 device to my domain.
Part 6: Creating Organizational Units (OUs), users, and groups, and demonstrating logging into accounts.
Part 7: Sharing folders securely within the AD environment.
